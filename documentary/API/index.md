
## API

The package is available by importing its default function:

```js
import vscodeRequireTypes from 'vscode-require-types'
```

```### vscodeRequireTypes
[
  ["arg1", "string"],
  ["arg2?", "boolean"]
]
```

Call this function to get the result you want.
